# 2026-04-02 Changelog

## Session 1: 系统初始化与核心功能建设

### 已完成功能（v1.0 基础版）

#### Skills
- [新增] `/read-paper` — 单篇论文深度阅读分析，支持 arxiv URL/ID/本地 PDF，`--repo` 关联代码，`--prompt` 自定义角度，Q&A 追问模式
- [新增] `/daily-papers` — 每日论文爬取与推荐，基于 interests.md 筛选，智能排序
- [新增] `/survey-topic` — 研究方向快速掌握，生成结构化综述
- [新增] `/analyze-code` — 代码仓库分析，可关联论文交叉分析
- [新增] `/manage-library` — 本地论文库管理（统计、搜索、列表、打标签、清理）
- [新增] `/session-digest` — 会话总结与归档，可选 skill 优化建议

#### 存储与数据结构
- [新增] `library/` 目录体系：papers/, tmp/, topics/, daily/, interests.md
- [新增] meta.md 三段式格式：概要总结 + Abstract 原文 + 摘要翻译
- [新增] 论文目录命名：`{arxiv_id}-{method_slug}`（如 `2604-01030-diff3r`）
- [新增] tmp → papers 收藏确认机制

#### 脚本
- [新增] `arxiv_fetch.py` — arXiv 论文元信息获取
- [新增] `arxiv_daily.py` — 批量论文列表抓取（HTML scraping + API fallback），支持 CSV 输出和 alphaxiv 批量详情获取

### 本次会话变更

- [修改] 论文目录名从 `{arxiv_id}` 改为 `{arxiv_id}-{method_slug}`，更直观
- [修改] meta.md 格式统一为三段式（概要总结 + Abstract + 摘要翻译）
- [修改] `arxiv_daily.py` 重写为两阶段流程（list → CSV, detail → JSON），减少交互确认次数
- [修改] daily-papers skill 流程优化，引入批量抓取和批量详情获取
- [修改] 所有 skill 中论文目录引用更新为新命名格式
- [迁移] `library/papers/2602-08169` → `library/papers/2602-08169-spherical-steering`
