# v1.0.6 变更日志 — 2026-04-21

## Arxiv ID 有效性验证

### 新功能
- `/read-paper` 新增 **Step 0b-1: Arxiv ID 有效性验证**——当本地找到论文 meta.md 时，自动验证 arxiv ID 是否仍指向同一篇论文
- 验证方法：用 `arxiv_fetch.py` 获取当前元信息，比对标题
- 标题不匹配时自动触发修正流程：搜索正确 ID → 确认 → 更新 meta.md、目录名、sources.md、analysis.md、daily 日志 → 刷新索引
- Step 1 的 `arxiv_fetch.py` 调用也增加交叉验证逻辑

### Bug 修复
- 修复 PoCo 论文 arxiv ID 错误（2604.03741 → 2604.03738），arxiv 版本更新导致 ID 重分配
- 修正 `library/daily/2026-04-07.md` 中的两处旧 ID 引用

### 动机
arxiv 论文在提交新版本时可能被重新分配 ID，导致本地记录的 ID 指向完全不同的论文。此前无验证机制，可能在分析时引用错误论文内容。新检查在每次读取本地论文时自动验证，防止此类问题。
